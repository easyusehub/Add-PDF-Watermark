# 💧 Add PDF Watermark | EasyUseHub

Free, client-side tool to **add custom text watermark** to PDF files. Fully browser-based, no API required, supports files up to **20MB**.

## 🚀 Features
- Add **custom text watermark**  
- Select **position**: Center, Top/Bottom Left/Right  
- Choose **color** and **font size**  
- Fully **client-side**, no API needed  
- Premium **ILovePDF-style UI**  

## 📂 How to Use
1. Open [Demo](https://easyusehub.github.io/Add-PDF-Watermark/add-watermark.html)  
2. Upload PDF → Enter watermark → Set position, color, size → Click **Add Watermark**  
3. Download processed PDF
